# Guia de Recomendações de Jogos

Um site simples para recomendar e catalogar jogos eletrônicos.

## Descrição

Este projeto consiste em uma página web estática que funciona como um guia ou blog de recomendações de jogos. O objetivo é, criar reviews rápidas, organizar jogos por gênero/plataforma. Os usuários podem navegar pelas recomendações e encontrar sugestões de títulos para jogar.

## Tecnologias Usadas

As seguintes tecnologias foram utilizadas no desenvolvimento deste projeto:

*   **HTML:** Utilizado para estruturar as diferentes seções do site (cabeçalho, listas de jogos, descrições e rodapé) de forma semântica.
*   **CSS:** Utilizado para estilizar a interface, definir o layout responsivo, cores, tipografia e garantir uma experiência visual agradável para quem busca recomendações de jogos.
